## ✨ Overview

**QuantixLauncher** is a powerful multi-instance Roblox launcher built for scale, automation, and control.

It integrates **QuantixCloud**, a **local API**, and **advanced automation tools** to allow users to run and manage unlimited Roblox sessions simultaneously.

---

## 🔥 Features

### ☁️ QuantixCloud
- Unlimited free accounts
- Instant access, no setup required
- Fully integrated with launcher

---

### ⚙️ Local API
- Full local control over launcher
- Build custom tools and automation
- Extend functionality easily

---

### 🧠 Xeno Functions
- Built-in execution functions
- Optimised for performance
- Designed for automation workflows

---

### 🎮 Multi-Instance Engine
- Launch unlimited Roblox instances
- No restrictions
- Stable multi-session handling

---

### 🔄 Account System
- Bulk cookie import
- Single cookie import
- Easy account switching

---

### 🔍 Game Search
- Search Roblox games directly
- Launch instantly from UI

---

### 🛠️ Tools
- Anti-AFK system
- Low graphics mode
- Optimised for long sessions

---

### 🔔 Updates
- Auto update notifications
- Always stay up to date

## 📦 Installation

```bash
git clone https://github.com/angussaw4134/mutiple-roblox-launcher.git
cd mutiple-roblox-launcher
npm install
npm start
```

---

## 🧩 Usage

1. Open QuantixLauncher  
2. Connect QuantixCloud (optional)  
3. Import accounts  
4. Search or select a game  
5. Launch multiple instances  
6. Control via UI or API  

---

## 🔌 Local API

```bash
GET /instances
POST /launch
POST /import-cookies
```
